# Описание
Этот скрипт на Python использует API ExchangeRate-API для получения актуальных курсов обмена для указанной валюты. Пользователь может указать валюту через аргументы командной строки, и скрипт вернёт курсы обмена этой валюты к другим валютам.

##  Как установить
Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2)

Для установки зависимостей:
```
pip install -r requirements.txt
```
Рекомендуется использовать virtual/venv

Перед запуском программы не забудьте склонировать проект или скачать его архив. В папке с проектом не забудьте создать .env файл, где вы укажите ваш собственный токен!

Чтобы создать токен заходите на сайт(https://www.exchangerate-api.com/), регистрируетесь и получайте apikey который записывайте в фаил .env в переменную TOKEN:

Пример: TOKEN=ваш токен

## Как запустить
Запустите скрипт, выполнив команду в терминале:

python main.py
