# Конвертер валют

Этот скрипт на Python использует API ExchangeRate-API для получения актуальных курсов обмена для указанной валюты. Пользователь может указать валюту через аргументы командной строки, и скрипт вернёт курсы обмена этой валюты к другим валютам.

##  Как установить

1. Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2)


2. Перед запуском программы не забудьте склонировать проект или скачать его архив. В папке с проектом не забудьте создать .env файл, где вы укажите ваш собственный токен!

```
git clone https://github.com/....../.git
cd название_проекта
```


3. Рекомендуется использовать [virtual/venv](https://www.nic.ru/help/kak-sozdat6-virtual6noe-okruzhenie-v-python_11757.html?utm_source=google.com&utm_medium=organic&utm_campaign=google.com&utm_referrer=google.com)

4. Для установки зависимостей:

```
pip install -r requirements.txt
```

5. Чтобы создать токен заходите на [сайт](https://www.exchangerate-api.com/), регистрируетесь и получайте apikey который записывайте в фаил .env в переменную TOKEN.Пример: 

```
TOKEN=ваш токен
```

## Как запустить

Запустите скрипт, выполнив команду в терминале:

```
python main.py
```
## Использование

Для запуска скрипта введите в командной строке:

  ```
  python main.py <base_cyrrency> <targeted_curency> <amount> 
  ```
## Переменные окружения
* <base_currency> — базовая валюта(например, "RUB").
* <target_currency> — целевая валюта (например, "USD").
* <amount> — число для конвертации  (например"1000").


Пример:

```
python main.py RUB USD 1000
```
иначе значения будут default
